## JS 面向对象
- JS数据类型
  {} 对象 object
  [] 数组
  字符串 '' "" String
  数值型 Number 21
  布尔类型 Boolean
- JS中面向对象有两种表达方式
  1. class 方式构造
  抽象的类，以及可以使用的对象的概念 constructor type n
  2. 对象字面量的方式来构建 { } 具有描述性
  属性+犯法的复杂数据类型
  对象.属性 对象.函数（target）
  - 代理模式 Proxy
    小明 ——> 小红   简陋
    小明 -> 小美  复杂，细腻
    不同的对象（小红，小美），实现相同的方法 （receiveFlower） 交换使用， 面向接口的编程
